iOS向けアプリケーションの作成

必要なもの

  • Intel-based Computer with Mac OS X Lion or greater(10.7.4++)
  • Xcode 4.5以降
  • iOS 6 SDK
  • Xcode Command Line Tools
  • iOS Device

インストール

XcodeをApp Storeからダウンロードしてインストールします。 Xcode 4.5.2以降をダウンロードした場合、iOS 6 SDKも含まれているため、それ以前のXcodeを使用している場合のみ、別途Apple Developer Downloadより入手してください。 iOS 6 SDKを入手するためには、Apple Developerに登録する必要があります。

Xcodeをインストール後、Xcode Command Line Toolsをインストールする必要があります。 Preferences -> Downloads -> Componentsを開き、Command Line ToolsのInstallをクリックします。

また、Preferences -> Locations -> Source Treesを開き、+ボタンをクリックし、Cordovaライブラリへのパスを追加します。
Setting NameとDisplayNameにVSD_CORDOVALIB、PathにVSDが保持しているCordovaLibへのパスを設定します。

VSDが保持しているCordovaLibへのパスは以下の通りです。 VSD.appを保存してあるフォルダまでのパスと合わせて記述してください。 VSD.app/Contents/Resources/plugins/extensions/PhoneGap/libs/ios/CordovaLib

読み込みと実行

File -> Openを選択し、エクスポートしたフォルダの”XXXXX.xcodeproj”を選択して、Openをクリックします。

左上に表示されるRunをクリックするとエミュレータが起動し、アプリケーションが実行されます。

実デバイス上で実行する

Apple Developerに登録し、デバイスにデプロイするために必要な手続きをiOS Provisioning Portalから行ってください。 詳細な手順に関しては、Tools Workflow Guide for iOSを参照してください。

正しく設定か完了すると、デバイスを選択することが出来ます（ここではiPadを登録しています）。

デプロイ先のデバイスを選択し、Runをクリックすると、デバイス上にアプリケーションがインストールされ、自動的に実行されます。
